## Versi 4.0.0 (Agustus 2025)

### Major Release: Glowar Design System

**Tanggal:** 17 Agustus 2025  
**Author:** Ardelyo

**Perubahan Besar:**

### Glowar Design Language

**Filosofi:** "Luminous Depth"
Bahasa desain baru yang menolak antarmuka datar demi lingkungan digital yang imersif dan kaya.

**5 Pilar Glowar:**
1. **Deep, Not Black:** Menggunakan 'Deep Charcoal' (`#0D0D0D`) sebagai kanvas utama, bukan hitam pekat.
2. **Functional Light:** Cahaya digunakan untuk hierarki dan fokus (Aurora Gradients).
3. **Glass & Layering:** Penggunaan glassmorphism untuk memisahkan layer visual.
4. **Soft Geometry:** Sudut membulat (16px-24px) dan ritme 8px grid.
5. **Fluid Motion:** Animasi berbasis GSAP feel untuk interaksi yang hidup.

**Implementation:**
- Redesign seluruh komponen UI menggunakan prinsip Glowar
- Migrasi dari Tailwind default colors ke custom palette
- Implementasi sistem animasi global

**Benefits:**
- Identitas visual yang unik dan premium
- Kenyamanan mata (reduced eye strain)
- Interaksi yang lebih intuitif

## Lesson Learned

### Architecture Decisions

**File-based Storage:**
- Pros: Faster, simpler, version controlled
- Cons: Requires rebuild untuk content updates
- Decision: Worth it untuk project scope

**Static First:**
- Pros: Better performance, simpler deployment
- Cons: Less dynamic features
- Decision: Aligns dengan portfolio nature

### Best Practices
- Performance budget enforcement penting
- Security scanning dari awal lebih baik
- Mobile-first prevents banyak rework
- Regular bug fix sprints maintain quality
