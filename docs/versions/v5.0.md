## Migration Guide

### From v4.x to v5.0

**Breaking Changes:**
- Color system completely revised
- Component APIs updated (minor)
- Background system changed

**Steps:**
1. Update color references
2. Check component props
3. Test animations
4. Verify responsive behavior
5. Validate accessibility

**Code Changes:**
```typescript
// v4.x
className="bg-background"

// v5.0
className="bg-[#030303]"
```
